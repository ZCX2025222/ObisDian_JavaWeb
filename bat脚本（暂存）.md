cls：终端清屏命令
.\：运行某程序命令


---

ECHO：回显命令
	ECHO ON/OFF - 开启/关闭回显功能
	ECHO 信息内容 - 输出提示信息
	ECHO 文件内容>>文件名 - 创建新文件并插入内容

@：写在命令前面，作用是关闭命令的回显，无论此时命令是否开启回显。

注释命令：
	rem 被注释的内容 - 注释命令，被注释的命令不会执行，但能回显。
	:: 被注释的内容 - 通常用作注释，不会回显。

PAUSE：暂停命令，运行到此命令时，会暂停执行命令，并显示：请按任意键继续...
	技巧：ECHO 其它提示语句 & pause > nul 
		效果：不显示提示语句，但仍需按下任意按键才能继续执行命令。

TITLE：设置批处理窗口的标题
	格式：title 新窗口标题

COLOR：设置批处理控制台的背景 和 字体 颜色
	格式：COLOR 2个16进制颜色格式，分别是 背景 和 字体 

Mode：设置批处理窗口大小
	格式：Mode con cols=高度 lines=宽度

Goto：跳转命令
	指定跳转到标签行，找到标签行后，程序将处理从下一行开始的命令。
	格式：Goto :标签名
```bat
echo 开始跳转。
goto :zcx

echo 这部分被跳过了，没有执行。

:zcx
echo 程序跳转到这儿了。

```

Find：文件搜索命令
	格式：Find "字符串" 要搜索的文件名
	注意：该搜索是模糊搜索，只要被搜索的文件含有该字符串，就会被搜索出来。
如：
```bat
find "111" 2.txt
```

Type：获取文件内容命令
	格式：Type 文件名
如：
```bat
Type 2.txt
```

START：启动命令
	作用：启动应用程序。
	格式：start 应用程序名
	可启动的程序有：
		calc --- 计算器
		notepad --- 记事本
如：
```bat
start calc
```

CALL：调用命令
	作用：CALL命令可以在批处理执行过程中调用另一个批处理，当另一个批处理执行完后，再继续执行原来的批处理。
	格式：call 调用目标
如：
```bat
call 2.txt --- 能打开文件
call calc  --- 能打开电脑程序
call :zcx  --- 能直接跳转到goto标签
```
