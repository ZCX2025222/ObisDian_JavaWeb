不认识的方法：
	字符串.trim()：去除字符串的开头和结尾的空格。
	XX.isEmpty()：判断XX是否为空，若是，则返回布尔值 true。

数据转换部分：
	注意：from表单提交的是字符串形式，如果数据库中的某些字段为非字符串型，需要提前将该字段的数据类型转换成字段的数据类型。
```java
// 将表示价格的字符串转换为双精度浮点数(double)  
double price = Double.parseDouble(priceStr);  
// 将表示数量的字符串转换为整数(int)  
int quantity = Integer.parseInt(quantityStr);
```

URL的分隔：
```java
//用?将左边的url和右边的action参数分隔开，并传递到url对应的java文件处。
action="/Book_Shopping/book_con?action=add"
//用&符号将action和id参数连接起来，并传递到url对应的java文件处。
<a href="book_con?action=detail&id=${book.bid}">查看详情</a>
```
