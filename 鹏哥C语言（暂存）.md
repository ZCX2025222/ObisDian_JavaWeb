# 指针
内存单元的编号 = 地址 = 指针 = 如：0x10101010

& --- 取地址操作符。
int* p --- 创建指针变量p，这里表示指针变量p指向的变量的数据类型为int型。
\*p --- 解引用操作符
指针的类型决定了指针向前或向后走一步有多大距离，类型步长。
void* --- 无具体类型（泛型）指针，可以接收任意类型的指针，但无法参与运算。
const --- 将变量修饰为常变量，具有常量属性，无法被修改。
```c
int n = 10;
int m = 100;
// const与谁相邻，谁就被限制。

// 情况1
int const *p = &n;
// const 修饰指针变量。
// 放在*的左边，限制的是指针指向的内容，也就是不能通过指针变量来修改它所指向的内容。
// 但是指针变量本身是可以改变的。

*p = 20; // 错误
p = &m;  // 正确

// 情况2
int* const p = &n;
// const 修饰指针变量。
// 放在*的右边，限制的是指针变量本身，也就是指针不能改变它的指向。
// 但是可以通过指针变量修改它所指向的内容。

*p = 20; // 正确
p = &m;  // 错误

// 情况3
int const * const p = &n;
// const 修饰指针变量。
// 放在*的两边，既限制了指针指向的内容，又限制了指针变量本身。
// 无法改变指针指向的内容和指针变量本身。 

*p = 20; // 错误
p = &m;  // 错误
```
