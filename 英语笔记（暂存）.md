# 名词
分为 普通名词 和 专有名词。
普通名词分为 可数名词 和 不可数名词。
	可数名词：指那些肉眼能分清数量的名词，比如：书、房子、手机。
		每个可数名词都有两种格式：一个是单数的格式，另一个是复数的格式（+s）。
	不可数名词：与可数名词相反，指肉眼分不清数量的名词：比如：大部分液体、头发。

名词的定义：世间万物的名词。
名词的分类：
	专有名词 = 人名、地名和其他专有名词。
	普通名词 = 可数名词 和 不可数名词。
普通名词：
	可数名词 = 字典规定可数：单数格式（book）和 负数格式（books）
	不可数名词 = 字典规定不可数 = 只有一种格式 = 没变化 
大多数可数名词的复数格式都是由单数格式变化而成，规则如下:
	1. 可数名词词尾直接加+s如：book-books
	2. 字母s、sh、x、ch结尾的可数名词+es 如：box-boxes
	3.  辅音字母+y结尾的可数名词，去 y 变 i+es 如：city=cities
	4.  字母 f 或 fe 结尾的可数名词，去 f 或 fe 变 v 再 +es 如：knife-knives
	5. 辅音字母 +o 结尾的可数名词，直接 +tes，如：potato-potatoes
当然，也有不少常见可数名词单数格式变复数格式是不规则的，比如：
	man-men     男人
	mouse-mice    老鼠
	goose-geese    鹅
	child-children    小孩
不可数名词只有单数格式。
不可数名词可以用量词来修饰。

名词的作用：
	1.作主语:The bookis new.
	2.作表语:He is my friend.
	3.作宾语:I like pears
	4.作宾补:I call him Uncle Tom.
	5.作前置定语:English book=也可以理解复合名词
	6.名词作介词宾语(暂不讲)
	7.名词作同位语(暂不讲)
